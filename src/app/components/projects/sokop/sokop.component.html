SOKOP

November, 2021 - April, 2022

Summary

SOKOP Mal Information system provides the employees of the court to enter, view, search and generate various court case documents. Users have role based and court level access to information and based on roles certain information and screens are dynamically rendered. Information system is designed to allow approval of many litigations in a single request based on user level authorization.

Role: Full stack development
Tasks:
  • Implementing data entry screens for managing court case information
  • Implementing REST services to support lookup value reference data.
  • Implementing administration screens for various types of reference data
